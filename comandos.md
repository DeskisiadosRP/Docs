# ⌨️ Comandos del Servidor

## 📋 Guía Completa de Comandos

Domina los comandos esenciales para una experiencia de roleplay óptima en Deskisiados RP.

***

## 1️⃣ /ME - Acciones del Personaje

{% hint style="info" %}
**Uso:** Describir acciones físicas que no tienen animación en el juego
{% endhint %}

### 📖 Descripción

El comando `/me` se utiliza para narrar las acciones físicas de tu personaje que no pueden ser representadas mediante animaciones del juego. Estas acciones deben ser detalladas y realistas.

### ✅ Ejemplos Correctos

```
/me saca su billetera desde su bolsillo.
/me observa nerviosamente a su alrededor.
/me se ajusta las gafas con cuidado.
/me toma un sorbo de su café.
```

### 💡 Mejores Prácticas

* ✏️ Sé descriptivo pero conciso
* 🎯 Mantén las acciones realistas
* ⏱️ Usa presente para mayor inmersión
* 🚫 No abuses del comando

***

## 2️⃣ /DO - Percepciones y Entorno

{% hint style="info" %}
**Uso:** Describir lo que tu personaje percibe, piensa o siente, y preguntar sobre el entorno
{% endhint %}

### 📖 Descripción

El comando `/do` permite expresar las percepciones, pensamientos, emociones de tu personaje, así como describir detalles del entorno que otros pueden observar. También se usa para hacer preguntas sobre el estado de cosas o personas.

### ✅ Ejemplos Correctos

```
/do Se siente cansado y agotado.
/do Sus manos estarían temblando visiblemente.
/do ¿Se ve alguna cámara de seguridad en el lugar?
/do ¿El sujeto tiene alguna herida visible?
```

### 💡 Cuándo Usar /DO

| Situación                | Ejemplo                                    |
| ------------------------ | ------------------------------------------ |
| **Emociones**            | `/do Se sentiría ansioso`                  |
| **Estado físico**        | `/do Estaría sudando nerviosamente`        |
| **Preguntas de entorno** | `/do ¿Hay personas alrededor?`             |
| **Detalles visibles**    | `/do Tendría manchas de sangre en la ropa` |

***

## 3️⃣ /ENTORNO - Alertas a Autoridades

{% hint style="danger" %}
**OBLIGATORIO:** Este comando DEBE usarse en situaciones delictivas. Su omisión resulta en sanciones graves.
{% endhint %}

### 📖 Descripción

El comando `/entorno` simula las llamadas que civiles, testigos o cámaras de seguridad harían a las autoridades (LSPD, Sheriff) al presenciar actividades delictivas. Es fundamental para mantener el realismo del servidor.

### 📝 Elementos Requeridos

Incluye SIEMPRE que sea relevante:

* 👥 **Cantidad de personas** involucradas
* 👔 **Descripción física:** Ropa, colores, características visibles
* 🔫 **Armamento:** Tipo y calibre (si es visible)
* 🚗 **Vehículo:** Marca, modelo, color
* 🧭 **Dirección de huida** o ubicación actual
* 👨‍👩‍👧 **Rehenes:** Cantidad y estado (solo si aplica)
* 📍 **Ubicación específica** del incidente

### ✅ Ejemplo Completo

{% hint style="success" %}
```
/entorno ¡Alo policía! Acá la señora Pía, me encuentro trabajando en Cubitos, 
se verían a dos sujetos vestidos totalmente de negro enmascarados, estarían 
portando armas de bajo calibre y se encuentran robando el banco Fleeca. Se 
estarían movilizando en un auto Sultan RS de color blanco y estarían huyendo 
en dirección norte. ¡Por favor ayuda!
```
{% endhint %}

### ⚠️ Consecuencias de No Usar /ENTORNO

| Infracción       | Sanción                   |
| ---------------- | ------------------------- |
| **Primera vez**  | Advertencia + Educación   |
| **Reincidencia** | Kick del servidor         |
| **Frecuente**    | Ban temporal o permanente |

## 4️⃣ /AUXILIO - Llamadas a EMS

{% hint style="info" %}
**Uso:** Solicitar ayuda médica cuando tu personaje está inconsciente y no hay nadie cerca
{% endhint %}

### 📖 Descripción

Comando para reportar emergencias médicas al servicio de EMS. **Siempre** debe escribirse en tercera persona, como si fuera un testigo reportando la situación.

### ✅ Ejemplo Correcto

```
/auxilio ¡Auxilio EMS! Estoy viendo a un sujeto tirado en la calle cerca del 
Burger Shot. Intenté buscar su pulso y este sería estable pero no reacciona. 
Por favor vengan a ayudarlo.
```

### 📋 Cuándo Usar

* ❤️ Tu personaje está inconsciente
* 👥 No hay otras personas cerca para ayudarte
* 🏥 Necesitas atención médica urgente

***

## 5️⃣ /REPORT - Soporte Técnico

{% hint style="warning" %}
**Uso Exclusivo:** Problemas técnicos, bugs o consultas urgentes al staff
{% endhint %}

### 📖 Descripción

Utiliza `/report` para reportar problemas técnicos al staff disponible en el servidor. **NO** debe usarse para consultas de roleplay.

### ✅ Usos Apropiados

* 🐛 Bugs del juego (quedarse atascado, items desaparecidos)
* 🗺️ Problemas de terreno (caerse bajo el mapa)
* ⚙️ Errores de mecánicas del servidor
* 🚨 Violaciones graves a la normativa
* ❓ Consultas técnicas urgentes

### ❌ Usos Inapropiados

* Preguntas sobre roleplay
* Quejas sobre otros jugadores sin evidencia
* Conversación casual con el staff
* Consultas que pueden esperar

***

## 6️⃣ /ANUNCIOS - Publicidad de Locales

{% hint style="info" %}
**Exclusivo:** Solo para dueños o encargados de locales comerciales
{% endhint %}

### 📖 Descripción

Comando para enviar anuncios publicitarios sobre tu local comercial: aperturas, promociones, cierres y eventos especiales.

### ✅ Ejemplo Correcto

```
/anuncios ¡Benny's Motor Works está abierto! Tenemos todo listo para ti. 
Mientras tu vehículo recibe el mejor servicio mecánico, puedes disfrutar 
de una buena bebida en nuestra barra. ¡Te esperamos!
```

### ⏱️ Regla de Tiempo

{% hint style="warning" %}
**Importante:** Debe haber un **mínimo de 10 minutos** entre anuncios por cada local.\
**Ejemplo: Si tu tienes un local y tiras un anuncio debes de esperar 10 minutos, esto es solo tu local, cada local o facción tiene su limite de 10 minutos independiente.**\
\
**Ejemplo 2: En el caso de que tu como EMS tiras un anuncio y UwU Cafe tira otro a los 2 minutos es valido ya que es un local distinto a EMS.**
{% endhint %}

### 📝 Tipos de Anuncios Permitidos

| Tipo          | Descripción                          |
| ------------- | ------------------------------------ |
| **Apertura**  | Anunciar que el local está abierto   |
| **Cierre**    | Informar que el local cerrará pronto |
| **Promoción** | Ofertas especiales o descuentos      |
| **Evento**    | Actividades especiales en el local   |

***

## 7️⃣ /AYUDA - Consultas Rápidas

{% hint style="danger" %}
**Uso Responsable:** Solo para consultas legítimas. El abuso será sancionado.
{% endhint %}

### 📖 Descripción

Comando para realizar **consultas importantes** dentro del servidor. No debe usarse para conversaciones innecesarias.

### ✅ Usos Válidos

* 🛠️ Preguntar por disponibilidad del staff
* ⌨️ Apoyo con teclas o configuración básica
* 💼 Consultas sobre trabajos (jobs)
* ❓ Dudas sobre mecánicas del servidor

### ❌ Usos Prohibidos

* Conversaciones casuales
* Bromas o trolleo
* Consultas sin relevancia
* Spam de mensajes

### ⚖️ Consecuencias del Mal Uso

| Infracción       | Consecuencia                    |
| ---------------- | ------------------------------- |
| **Primera vez**  | Advertencia verbal              |
| **Reincidencia** | Advertencia formal              |
| **Abuso**        | Suspensión temporal del comando |
| **Abuso grave**  | Sanción administrativa          |
